# NetEF-Blog-Api
Backend api with .NET EF Core and Mysql database for the vue-blog

# Run instructions
## Clone the repo
### $ git clone https://github.com/titusowuor30/NetEF-Blog-Api.git
### $ cd NetEF-Blog-Api
### $ dotnet build
### $ dotnet run

# Requirements
## Install <a href="https://dotnet.microsoft.com/download">.NET 5 SDK</a>
